# Grimrold v9.5 — Полная структура
## API:
- POST /ask { message }
- POST /code { code }

## Установка:
1. Двойной клик install.bat
2. Переход по адресу http://localhost:3000

## Структура:
- core/ — интеллект, память
- plugins/ — модули
- agents/, mutate/, logs/, ui/
- models/ — сюда положи .gguf-модели
